# Judge Quickstart

This guide is intended for a fast and consistent demo review experience.

## 1) Start Here

- Open the live demo
- Review platform brief and architecture overview
- Navigate readiness and report screens

## 2) What to Evaluate

- Clarity of readiness scoring output
- Practicality of prioritized findings
- Executive readability of generated report narrative
- Product completeness and deployment readiness

## 3) Recommended Demo Path

1. Overview of problem and solution
2. Walkthrough of readiness input and scoring result
3. Show architecture and trust-boundary approach
4. Present executive report output and action plan

## 4) Notes on Demo Integrity

- Demo mode is read-only for consistency
- Data is synthetic and presentation-safe
- Scoring is deterministic; AI is narrative-only
